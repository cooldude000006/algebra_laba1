* Сравнение методов решения СЛАУ (Лабораторная работа №1)

** Описание
Данная работа реализует сравнительный анализ методов решения систем линейных алгебраических уравнений (СЛАУ):
- Метод Гаусса без выбора ведущего элемента.
- Метод Гаусса с выбором главного элемента по столбцу.
- LU-разложение с прямым и обратным ходом.

** Требования
- Python 3.8+
- Библиотеки: `numpy`
